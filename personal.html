🔥 ตั้งค่า Firebase Realtime Database
🔥 วิธีตั้งค่า Firebase (ทำครั้งเดียว)
1. ไปที่ console.firebase.google.com
2. สร้างโปรเจกต์ใหม่ → เลือก Realtime Database → Create database
3. เลือก Start in test mode (อ่าน/เขียนได้โดยไม่ต้อง login 30 วัน)
    หรือตั้ง Rules เป็น "read":true,"write":true ถาวร
4. คัดลอก Database URL รูปแบบ: https://xxx-default-rtdb.firebaseio.com
5. ไปที่ Project Settings → General → Web API Key → คัดลอก API Key
ไม่ต้องใส่ / ต่อท้าย
Project Settings → General → Web API Key
ชื่อ root node ใน database (ค่าเริ่มต้น: aps_data)

☁️ ตั้งค่า Google Drive Upload
📋 วิธีตั้งค่า (ใช้ Apps Script เดียวกับ Firebase)
1. ใช้ Apps Script Web App URL เดิมที่ deploy ไว้แล้ว
2. กรอก Google Drive Folder ID ที่ต้องการเก็บไฟล์
   (จาก URL: drive.google.com/drive/folders/[ID])
3. ตรวจสอบว่าในไฟล์ Code.gs
   แก้ var FOLDER_ID = 'YOUR_FOLDER_ID' ให้ถูกต้อง
URL เดียวกับที่ใช้ใน Firebase Settings
ดู Folder ID จาก URL ของโฟลเดอร์ใน Drive
